## Summary

These configuration files are built using the [dotbot](https://github.com/anishathalye/dotbot) repo for bootstrap installation. In addition, it also uses Homebrew's `Brewfile` in conjunction with the `brew bundle` command in order to install CLI tools and applications from both brew and the Mac App Store.

## Installation
Clone this repo to `$HOME/.dotfiles` with `git clone https://github.com/mattbeiswenger/dotfiles.git .dotfiles`
Run `./install.sh`
